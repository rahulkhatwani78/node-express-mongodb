# Node, Express, and MongoDB Learning Modules

This repository contains documentation and practical code covering Node.js, Express, and MongoDB.

### Modules

*   **[001: Node Internals, Event Loop & Thread Pool](./001-nodeInternals-eventLoop-threadPool.md)**  
    Overview of Node.js internals, diving deep into the event loop and thread pool architecture.

*   **[002: Streams & fs Module](./002-streams-fsModule.md)**  
    Working with readable, writable, and transform streams, and managing files using the `fs` module.

*   **[003: Buffers, Worker Threads & Cluster](./003-buffer-workerThread-cluster.md)**  
    Handling raw binary data with buffers, understanding worker threads, and utilizing the `cluster` module for scaling.

*   **[004: HTTP Servers, URLs, Methods & Package Versioning](./004-httpServer-urls-methods-packageVersioning.md)**  
    Creating core HTTP servers, routing URLs, handling different HTTP methods, and understanding npm semantic versioning.

*   **[005: Express, REST API, Middleware, HTTP Headers & Status Codes](./005-express-restApi-middleware-httpHeaders-httpStatusCodes.md)**  
    Building robust RESTful APIs with Express.js, implementing middleware patterns, and mastering HTTP headers and status codes.

*   **[006: Week 3 Revision & Practical Tasks](./006-week3-revision-practicalTasks.md)**  
    Comprehensive review and practical implementations of Node.js and Express modules.

*   **[007: MongoDB Shell](./007-mongodb-shell.md)**  
    Introduction to MongoDB fundamentals and interacting with the database using the Mongo Shell.

*   **[008: Mongo in Node, MVC Architecture & Mongo Indexing](./008-mongoInNode-mvcArchitecture-mongoIndexing.md)**  
    Integrating MongoDB via Mongoose in Node.js applications, structuring projects with MVC architecture, and applying secondary/compound indexes.

*   **[009: MongoDB Aggregations](./009-mongoDBAggregations.md)**  
    Deep dive into the MongoDB aggregation framework and building advanced data processing pipelines.

*   **[010: Aggregation Practice & Query Planner](./010-aggregationPractice-queryPlanner.md)**  
    Advanced practical exercises using MongoDB aggregations and understanding query optimization using `explain()` and the Query Planner.

*   **[011: Transactions & ACID Properties](./011-transactions-acidProperties.md)**  
    Implementing MongoDB multi-document transactions and ensuring ACID compliance.

*   **[012: Replication & Sharding](./012-replication-sharding.md)**  
    Deploying MongoDB clusters via replica sets for high availability and sharding for horizontal scaling.

*   **[013: Week 4 Revision](./013-week4-revision.md)**  
    Overview and summary revision of MongoDB concepts including architectures, indexing, aggregations, and scaling.  
    *(Includes [Week 4 Practical Task Directory](./013-week4-practicalTask))*

*   **[014: EJS, Multer, WebSocket, REST APIs, HTTP APIs, GraphQL, RPC & gRPC](./014-eJS-multer-webSocket-restAPIs-HTTPAPIs-graphQL-RPC-gRPC.md)**  
    Extended curriculum incorporating EJS templating, Multer file uploads, real-time WebSockets, and alternative API designs (GraphQL, RPC, gRPC).

*   **[015: GraphQL](./015-graphQL.md)**  
    Deep dive exploring GraphQL core concepts: schemas, queries, and mutations.

*   **[016: Week 5 Revision](./016-week5-revision.md)**  
    Extensive revision of integrating diverse API architectural patterns, templating, and WebSockets.
